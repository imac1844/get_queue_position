# get_queue_position
Скрипт для запуска в терминале с выводом на экран информации о
мастерноде.
Запуск:
get_queue_position_pubkey.sh 119.217.7.183
Вывод на экран:
masternode 119.217.7.183 - RfgTijKj
      in queue position 4404  /  4739 
      6days 22h31m left till SELECTION!
[###-------------------------------------|SELECTION POOL]
   ^ 7.85%
